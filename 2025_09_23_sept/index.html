2. Objeto Math

En JavaScript, el objeto Math es una librería cuyos métodos son las clásicas funciones matemáticas, por ejemplo:

Ejemplo básico

document.write("<h1>", Math.random(), "</h1>")

mostraría un número aleatorio comprendido entre 0 y 1 (en Java, equivaldría a la clase java.lang.Math).

Nota: Realmente, el número 1 no saldrá nunca, ya que los números aleatorios se generan en el intervalo [0, 1).

3. Funciones del objeto Math

El objeto Math tiene multitud de funciones estáticas para realizar cálculos matemáticos. Algunos de ellos son:

Principales métodos y propiedades

Math.random() – número aleatorio entre 0 y 1

Math.ceil(x) – redondea al entero superior

Math.floor(x) – redondea al entero inferior

Math.round(x) – redondea al entero más próximo

Math.trunc(x) – trunca (elimina la parte decimal)

Math.max(x, y, z, ...) – devuelve el máximo de los argumentos

Math.min(x, y, z, ...) – devuelve el mínimo de los argumentos

Math.PI – 3.141592653589793

Objeto Date

En JavaScript, el objeto Date se utiliza para instanciar objetos que almacenan una fecha y hora. Por ejemplo:

Ejemplo básico

document.write("<h1>", new Date(), "</h1>")

mostraría la fecha y hora actual (en Java, equivaldría a la clase java.util.Date).

Date en MDN

Objeto String

En JavaScript, podemos encerrar una cadena de caracteres entre comillas simples ('), comillas dobles (") y acentos graves (`).

String en MDN

Ejercicios

a) ¿Qué ventaja tienen los template strings respecto a las otras formas de crear cadenas?

Permiten insertar expresiones y variables directamente dentro de la cadena con ${...}, además de mantener saltos de línea sin necesidad de concatenar.

b) ¿Qué método utilizas para obtener el 5º carácter de una cadena? (piensa en Java)

Se puede usar charAt(4) o acceder como un array con str[4].

c) ¿Para qué se utiliza el método indexOf(n)? (piensa en Java)

Para obtener la posición de la primera aparición de un carácter o subcadena dentro del string; devuelve -1 si no se encuentra.

d) ¿Qué variable miembro almacena la longitud de la cadena? (piensa en Java)

La propiedad length del string.

e) ¿Qué operador se utiliza para encadenar varios strings? (piensa en Java)

El operador +.

f) Comparación de strings con operadores:

Se pueden comparar con ==, ===, <, >, etc. "gato" es mayor que "Gato" porque las mayúsculas tienen menor valor Unicode que las minúsculas.
